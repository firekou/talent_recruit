# S1 — LinkedIn 直攻法

**策略代號：** S1  
**觸發點：** 主動搜尋目標候選人  
**主動性：** ★★★★★  
**適用場景：** 精準定義 ICP 後，主動出擊鎖定名單  

---

## 策略邏輯

```
定義目標候選人條件（TCP）
     ↓
LinkedIn / Sales Navigator 搜尋篩選
     ↓
Apify 抓取 + 情報蒐集（職涯背景分析）
     ↓
Talent-Scout Agent 候選人類型定位
     ↓
個人化連線邀請
     ↓
三封信序列（暖身 → 共頻 → 引入機會）
     ↓
Asana 記錄候選人狀態
```

---

## TCP（Target Candidate Profile）篩選條件

### 基本篩選

**職稱關鍵字：**
- 業務開發：Business Development Manager / BizDev / 業務經理 / 客戶開發
- 產品：Product Manager / PM / 產品經理 / 產品負責人
- 工程：Software Engineer / 後端工程師 / AI Engineer / Full Stack
- 行銷：Marketing Manager / Growth / 行銷企劃 / 內容行銷

**工作年資：** 3–10 年（有足夠經驗但仍有成長動機）

**地區：** 台灣（台北 / 新北 / 桃園 / 新竹）

**語言：** 中文為主，英文加分

### 加分篩選（優先觸達）

- [ ] 最近 3 個月更新過 LinkedIn 個人資料
- [ ] 有在 LinkedIn 發文分享工作心得或產業觀察
- [ ] 有取得新認證或完成課程
- [ ] 目前公司有新聞（裁員、縮編、併購）
- [ ] 目前任職公司規模 < 50 人（可能渴望更大舞台）

---

## 情報蒐集框架（每位候選人接觸前必做）

### 必看資訊

- [ ] LinkedIn About 段落（自我定位、語氣、關鍵字）
- [ ] 過去三個職位的發展軌跡（垂直晉升？跨域轉型？創業？）
- [ ] 最近 30 天的貼文與互動內容
- [ ] 目前公司規模、成立時間、最近動態
- [ ] 有無共同連線（可以提及）

### 職涯邏輯判斷問題

1. 他的職涯軌跡是「往上爬」還是「往外擴」還是「在重新定義自己」？
2. 他目前的職位對他而言是終點還是跳板？
3. 他最近在關注什麼主題？
4. 他的公司現在處於什麼階段？（成長？停滯？收縮？）
5. 他有沒有可以辨識的「換工作訊號」？

## 執行 SOP

### Step 1：建立目標名單（每週一）

```bash
# 執行 Apify 抓取
python scripts/apify_linkedin_talent.py --role "Business Development" --location "Taiwan" --limit 50

# 輸出：candidate-drafts/targets_[date].json
```

### Step 2：候選人分析與話術生成

對每位候選人，呼叫 `Talent-Scout` Agent：
- 輸入：LinkedIn 資料摘要
- 輸出：候選人類型 + 三封信序列

### Step 3：連線邀請（每日上限 20 人）

- 使用 Expandi 或手動發送
- 話術見 `05-talk-scripts.md` → S1 章節

### Step 4：三封信序列追蹤

| 時間點 | 動作 | 話術 |
|--------|------|------|
| Day 1（連線接受後） | 第一封私訊 | 暖身，不提職缺 |
| Day 5–7 | 第二封（有/無回覆版） | 共頻，自然帶入 |
| Day 8–12 | 第三封 | 引入機會 |

### Step 5：更新 Asana

- 每個候選人建立 Task
- 標記來源：`S1-Direct`
- 記錄每次接觸時間戳記

## 每日配額建議

| 動作 | 每日上限 | 備註 |
|------|---------|------|
| 連線邀請 | 20 人 | LinkedIn 官方限制 ~100/週 |
| 私訊（已連線） | 30 則 | 避免觸發 LinkedIn 限制 |
| Profile 瀏覽 | 100 人 | 有助於對方注意到你 |

## 成效追蹤指標

| 指標 | 目標值 | 說明 |
|------|--------|------|
| 連線接受率 | > 30% | 低於此需優化邀請話術 |
| 訊息回覆率 | > 15% | 低於此需優化第一封 |
| 面試轉化率 | > 5% | 從觸達到面試 |
| 週觸達人數 | 50–100 人 | 穩定規模 |
